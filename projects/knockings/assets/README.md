# Project assets

Drop everything this project needs in here:

- `thumb.jpg` — card thumbnail (recommended 800×800 or 1:1)
- `logo.glb` / `logo.svg` — turntable model / fallback logo shown in the card
- `<any>.glb` — 3D models for grid4-3d / grid4-phones blocks
- `<any>.mp4` — videos for grid4-videos-ppong and outro-video
- `<any>.png|jpg` — composite images, sketches, logo evolutions

Reference these files from `project.js` using paths like:
`projects/<this-slug>/assets/<filename>`
